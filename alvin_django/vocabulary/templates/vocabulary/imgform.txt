Got it — you want a simple HTML form where the user enters an image URL, submits it, and the same page displays that image using JavaScript (without reloading).
Here’s a complete, runnable example with input validation and error handling:

Html<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Image URL Loader</title>
<style>
    body { font-family: Arial, sans-serif; margin: 20px; }
    form { margin-bottom: 15px; }
    img { max-width: 400px; max-height: 400px; display: block; margin-top: 10px; }
    .error { color: red; }
</style>
</head>
<body>

<h2>Enter Image URL</h2>
<form id="imageForm">
    <input type="url" id="imageUrl" placeholder="https://example.com/image.jpg" required>
    <button type="submit">Show Image</button>
</form>

<div id="message" class="error"></div>
<img id="preview" alt="Image preview will appear here">

<script>
document.getElementById('imageForm').addEventListener('submit', function(e) {
    e.preventDefault(); // Prevent page reload

    const urlInput = document.getElementById('imageUrl').value.trim();
    const message = document.getElementById('message');
    const preview = document.getElementById('preview');

    message.textContent = ''; // Clear previous messages
    preview.src = ''; // Clear previous image

    // Basic URL validation
    try {
        new URL(urlInput);
    } catch {
        message.textContent = 'Please enter a valid URL.';
        return;
    }

    // Create a temporary image to check if it loads
    const testImg = new Image();
    testImg.onload = function() {
        preview.src = urlInput;
    };
    testImg.onerror = function() {
        message.textContent = 'Could not load image. Please check the URL.';
    };
    testImg.src = urlInput;
});
</script>

</body>
</html>


How it works:

Form submission is intercepted with preventDefault() so the page doesn’t reload.
URL validation is done using the URL constructor.
A temporary Image() object checks if the image can be loaded before displaying it.
If valid, the image is shown in the <img> tag; otherwise, an error message appears.

2. View (views.py)
Pythonfrom django.shortcuts import render
from django.core.validators import validate_email
from django.core.exceptions import ValidationError

def contact_view(request):
    context = {}
    
    if request.method == "POST":
        name = request.POST.get("name", "").strip()
        email = request.POST.get("email", "").strip()
        message = request.POST.get("message", "").strip()

        # Basic validation
        if not name or not email or not message:
            context["error"] = "All fields are required."
        else:
            try:
                validate_email(email)  # Validate email format
                # Here you could save to DB, send email, etc.
                context["message"] = "Your message has been sent successfully!"
            except ValidationError:
                context["error"] = "Invalid email address."

    return render(request, "contact.html", context)

How It Works

No Django forms — the form is pure HTML.

Validation is done manually in the view.
You can save to the database, send an email, or process data inside the view.

Got it — you want an HTML form where the user enters some data, and the page generates a URL based on that input.
Here’s a complete, runnable example that takes a text input and dynamically creates a clickable URL.

HTML + JavaScript Example
Html<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<title>Form to Create URL</title>
<style>
    body { font-family: Arial, sans-serif; margin: 20px; }
    label { display: block; margin-bottom: 8px; }
    input, button { padding: 8px; margin-bottom: 12px; }
    #result a { color: blue; text-decoration: underline; }
</style>
</head>
<body>

<h2>Create a URL from Input</h2>

<form id="urlForm">
    <label for="username">Enter username:</label>
    <input type="text" id="username" name="username" required>
    <button type="submit">Generate URL</button>
</form>

<div id="result"></div>

<script>
document.getElementById('urlForm').addEventListener('submit', function(e) {
    e.preventDefault(); // Prevent form from reloading the page

    const username = document.getElementById('username').value.trim();

    // Basic validation
    if (!username) {
        alert("Please enter a username.");
        return;
    }

    // Encode to make it URL-safe
    const encodedUser = encodeURIComponent(username);

    // Example: create a profile URL
    const generatedUrl = `https://example.com/profile?user=${encodedUser}`;

    // Display clickable link
    document.getElementById('result').innerHTML =
        `Generated URL: <a href="${generatedUrl}" target="_blank">${generatedUrl}</a>`;
});
</script>

</body>
</html>


How It Works

User enters text (e.g., john_doe).
On submit, JavaScript:

Validates the input.
Encodes it for safe URL usage.
Builds a URL like:https://example.com/profile?user=john_doe




Displays it as a clickable link.
