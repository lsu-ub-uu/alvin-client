ListSets

{% translate 'Hämtar arkivets set-struktur som möjliggör gruppering av poster i delmängder. Poster från Alvin kan hämtas i begränsade set utifrån ägande institution.' %}

GetRecord

https://www.alvin-portal.org/oai/oai?verb=GetRecord&identifier=oai:ALVIN.org:1&metadataPrefix=oai_dc

{% translate 'Hämtar en enskild, individuell metadatapost. Obligatoriska parametrar anger identifikatorn (identifier) och formatet (metadataPrefix) för de metadata som ska inkluderas i posten.' %}

{% translate 'En post (record) är metadata uttryckta i ett enda format. Den identifieras entydigt genom kombinationen av den unika identifikatorn för det objekt från vilket posten är tillgänglig, metadataprefixet som identifierar postens metadataformat samt postens datumstämpel. XML-kodningen av poster är organiserad i följande delar: ' %}

  1. header - {% translate 'innehåller den unika identifikatorn för objektet samt egenskaper som krävs för selektiv hämtning. Den består av följande delar:' %}
    • identifier - {% translate 'den unika identifikatorn för ett objekt i ett arkiv' %}
    • datestamp - {% translate 'datumet för skapande eller ändring av posten, att användas för selektiv hämtning.' %}
    • setSpec - {% translate 'set som posten ingår i, att användas för selektiv hämtning.' %}
  2. metadata - {% translate 'en enda manifestation av metadata från ett objekt. Den innehåller ett enda rotelement, til exempel oai_dc:dc, med underelement som tillhör motsvarande metadataformat.' %}

ListRecords

https://www.alvin-portal.org/oai/oai?verb=ListRecords&metadataPrefix=oai_dc

{% translate 'Hämtar fullständiga metadataposter från arkivet. Om antalet hämtade poster överstiger 100 returneras en lista över de första 100 posterna + ett resumptionToken-element i slutet av svaret som används för att fortsätta hämtningen av den kompletta listan. Inga andra parametrar kan då användas. Om resumptionToken saknas är metadataPrefix en obligatorisk parameter.' %}
